## ⚙️ 기술적 배경

### 과열 파라미터란?

김째 S2 매매법의 과열 감량은 **조건부 메커니즘**:

```python
# 기본 조건: QQQ MA3 > MA161 (또는 200MA 히스테리시스)
if q_ma3 > q_ma161:  # 기본 신호
    baseCode = 1 또는 2
    
    # 과열 감량은 baseCode == 2일 때만 작동
    if baseCode == 2:
        # dist200 (TQQQ vs 200MA 거리)에 따라 4단계 감량
        if dist200 >= s4_enter: stage = 4  # 최대 감량
        elif dist200 >= s3_enter: stage = 3
        elif dist200 >= s2_enter: stage = 2
        elif dist200 >= s1_enter: stage = 1
        else: stage = 0
```

**의미**:
- 과열 파라미터는 강한 신호(baseCode=2)에만 작동
- 약한 신호에서는 무시됨
- MA crossover가 기본 전제

## 🚨 주의사항

### 1. 과열 파라미터는 조건부
- **기본 전제**: QQQ MA3/161 crossover
- **작동 시점**: baseCode == 2 (강한 신호)만
- **약한 신호에서는 무시됨**

### 2. TQQQ 최적은 과적합
